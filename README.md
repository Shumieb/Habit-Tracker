✅ Habit Tracker Web Application

A full-stack web application to help users build, track, and maintain positive habits with real-time syncing, progress tracking, and interactive dashboards.

🚀 Features

🔐 Authentication – Secure login and signup with Firebase Auth

📅 Habit Management – Add, edit, delete, and track daily/weekly habits

🔄 Real-Time Syncing – Data stored and synced instantly using Firebase Firestore

🔥 Streak Tracking – Track consistency and visualize progress over time

📊 Analytics Dashboard – Progress insights and charts for motivation

🌐 Responsive UI – Built with Tailwind CSS and custom CSS

🚀 Deployment – Hosted on Firebase Hosting for fast and reliable delivery

🛠️ Tech Stack

Frontend: React (TypeScript), React Router v7, Tailwind CSS, CSS, React Icons

State Management: Zustand

Backend & Database: Firebase Firestore

Authentication: Firebase Auth

Deployment: Firebase Hosting
